# Sales Data Analysis

Sales analysis project built with Python, Pandas, and Matplotlib. The project
reads sales records from a local CSV file, calculates useful business metrics,
prints a summary in the terminal, and displays monthly revenue in a line chart.

The purpose of this project is to practice data manipulation, aggregation, and
visualization with a simple sales dataset.

## Features

- Reads sales data from a CSV file
- Calculates the total number of sales
- Calculates total revenue
- Calculates the average ticket value
- Identifies the best-selling product
- Identifies the product with the highest revenue
- Identifies the top-selling category
- Identifies the highest-spending customer
- Groups revenue by month
- Displays monthly revenue in a line chart
- Uses paths relative to the project folder

## How to Run

### 1. Open the project folder

```bash
cd Sales_Data_Analysis
```

### 2. Create a virtual environment

```bash
python -m venv .venv
```

Activate the virtual environment on Linux or macOS:

```bash
source .venv/bin/activate
```

On Windows:

```bash
.venv\Scripts\activate
```

### 3. Install the dependencies

```bash
pip install pandas matplotlib
```

### 4. Run the analysis

```bash
python analise_vendas.py
```

The results will be printed in the terminal, and a window containing the
monthly revenue chart will open.

## Dataset

The `vendas.csv` file stores the sales records used by the analysis. It
contains the following columns:

- `Data`: date of the sale
- `Pedido`: order number
- `Cliente`: customer name
- `Categoria`: product category
- `Produto`: product name
- `Quantidade`: quantity sold
- `Preco_Unitario`: price per unit

The revenue for each sale is calculated with:

```text
Revenue = Quantity × Unit Price
```

## How It Works

The main flow is handled by `analise_vendas.py`.

The program:

1. Locates and reads `vendas.csv` with Pandas.
2. Processes the dataset through separate analysis functions.
3. Calculates the main sales indicators.
4. Converts the date column to a date format.
5. Groups total revenue by month.
6. Prints the results in the terminal.
7. Uses Matplotlib to display the monthly revenue chart.

## Main Analyses

- Total sales: counts the number of records in the dataset.
- Total revenue: sums quantity multiplied by unit price.
- Average ticket: divides total revenue by the number of sales records.
- Best-selling product: compares the total quantity sold for each product.
- Highest-revenue product: compares the revenue generated by each product.
- Top category: compares the total quantity sold by category.
- Top customer: compares the total amount spent by each customer.
- Monthly revenue: groups revenue using the month of each sale.

## Chart

The monthly revenue chart uses:

- A line to represent revenue over time
- Markers to highlight each month
- A grid to make values easier to read
- Labels for the month and revenue axes

The `charts/` folder is used to organize image files created from the
analysis, such as `monthly_revenue.png`.
